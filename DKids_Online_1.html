⚙ Configurar base de datos Firebase
Sigue estos pasos una sola vez para conectar la app a internet y compartirla con todos los trabajadores. Es gratis.
1
Entra a console.firebase.google.com con tu cuenta Google y haz clic en "Agregar proyecto". Ponle nombre (ej: dkids-arriendos) y desactiva Google Analytics. Crea el proyecto.
2
Dentro del proyecto, haz clic en el icono </> (Web) para registrar una app web. Dale nombre (ej: DKids App). Copia el objeto firebaseConfig que aparece.
3
Ve a Firestore Database en el menu izquierdo → "Crear base de datos" → modo Produccion → elige la region us-central1.
4
En Firestore, ve a la pestana Reglas y reemplaza todo por:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
Luego haz clic en Publicar.

Para compartir con trabajadores: Despues de configurar, publica la app con Firebase Hosting:
1. Instala Node.js en tu PC → 2. Abre terminal en la carpeta del archivo HTML → 3. Ejecuta npm install -g firebase-tools → 4. firebase login → 5. firebase init hosting → 6. firebase deploy
Te dara una URL tipo tu-proyecto.web.app para compartir con todos.
